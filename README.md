## 📌 스터디 방식

- 스터디는 `매주 화요일 오후 9시`에 진행한다.
- 같이 읽을 책 한 권을 정하고 그 주에 읽을 챕터를 정한다.
- 책을 읽고 스터디 로그에 **①인상 깊었던 부분 ②느낀점**을 작성한다. 추가적으로 적고 싶은 부분은 알아서 추가한다.
- 스터디 날에 위에 작성했던 부분을 공유한다.
